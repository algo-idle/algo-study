# 2609번: 최대공약수와 최소공배수 (브론즈1)

| 시간 제한 | 메모리 제한 |
| :-------: | :---------: |
|    1초    |   128 MB    |

## 문제

두 개의 자연수를 입력받아 최대 공약수와 최소 공배수를 출력하는 프로그램을 작성하시오.

## 입력

```
24 18
```

## 출력

```
6
72
```
